本轮游戏规则如下：

（1）角色分配：你和另外四位随机匹配的参与者组成一组（共5人），每人分别扮演一个固定角色：A、B、C 、D或E。

（2）初始代币：每位参与者在每轮都拥有{{ C.ENDOWMENT }}初始代币。

{% if current_logic == 1 %}
（3） 角色任务：

如果你的角色为A、B、C 或 D，你需要独立决定，把多少代币（0–{{ C.ENDOWMENT }}之间任意整数）投入一个“公共池”，且剩余代币归自己所有。

如果你的角色为E，你需要观看A、B、C和D的决策之后，决定花多少代币增加/减少其中一人或多人的代币数，比率为1：3（即你付出1个代币，对方增加/减少3个代币）。

{% else %}
（3） 角色任务：

如果你的角色为A、B、C 或 D，你需要独立决定，如何把你的钱分配给公共池和转给玩家E，且剩余代币归自己所有。

如果你的角色为E，你需要在观察 A、B、C、D 的决策后，对每位玩家的分配逐一做出决策：

① 若接受某位玩家的分配：你不能减少该玩家的代币，且必须为该玩家增加代币（具体数量自行决定），增加比率为1 : 3（即你付出1个代币，对方获得3个代币）。

② 若拒绝某位玩家的分配：你可以选择是否增加/减少该玩家（或多位玩家）的代币，比率仍为1 : 3（即你付出1个代币，对方增加/减少3个代币）。

{% endif %}

（4）公共池收益：无论参与者投入多少，投入公共池的总金额将被翻一倍（即乘以2）后平均分配给小组内所有成员。

{% if current_logic == 1 %}
（5） 游戏收益公式：（本公式无需记忆，理解即可）

A、B、C 或 D = 你剩余的代币数 + 公共池收益 ± E增加/减少你的代币数

E = 你剩余的代币数

例如：A投入公共池1个代币，剩下19个；小组公共池总投入10个，翻倍后每人分得5个。 且E若花费1个代币以1:3的比例减少A的3个代币，则A最终获得19 + 5 - 3 = 21个代币，而E自己剩下19个代币作为最终收益。

{% else %}
（5） 游戏收益公式：（本公式无需记忆，理解即可）

A、B、C 或 D = 你剩余的代币数 + 公共池收益 ± E增加/减少你的代币数

E = 你剩余的代币数＋A、B、C 和 D分配给你的代币数

例如：A向公共池投入1个代币，并分配1个代币给E，自己剩余18个；小组公共池总投入10个代币，翻倍后每人分得5个。 E接受A的分配，不减少A的代币，且花费1代币以1:3的比例给A增加3个代币，因此A最终收益为18 + 5 + 3 = 26个代币。而E最终获得自己剩下19个代币加上A分配的1个代币，共20代币。

{% endif %}
